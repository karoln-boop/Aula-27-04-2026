Descrição do Git

O GitHub é a maior plataforma de hospedagem de código-fonte e arquivos em nuvem do mundo, utilizando o sistema Git para controle de versão. Ele funciona como uma rede social para desenvolvedores colaborarem, armazenarem, revisarem e gerenciarem projetos de software, permitindo trabalho em equipe e código O trabalho colaborativo, uma das características fundamentais do GitHub, é possível graças ao software de código aberto Git, que serviu como base para a criação do GitHub.
